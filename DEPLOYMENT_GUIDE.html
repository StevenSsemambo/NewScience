Overview
What You Have
Your MathQuest app is a React web app. Here's your deployment package:
📁 mathquest-pwa/ (your project folder)
mathquest-pwa/ ├── public/ │ ├── index.html ← App entry point │ ├── manifest.json ← Makes it installable │ ├── service-worker.js ← Offline support │ └── icons/ │ ├── icon-192.png ← App icon │ └── icon-512.png ← App icon (large) ├── src/ │ ├── index.js ← React bootstrap │ └── App.jsx ← Your full app └── package.json ← Project config
💡 You don't need to understand all these files. Just follow the steps below and your app will be live in under 15 minutes.
Step 1
Create a GitHub Account & Upload Files
GitHub stores your code online. It's free and required for Vercel deployment.
1
Create a GitHub account
Go to github.com → Click Sign Up → Use your email → Verify and log in.
2
Create a new repository
Click the + button (top right) → New Repository → Name it mathquest-ug → Set to Public → Click Create Repository.
3
Upload your files
On your new repo page, click uploading an existing file → Drag and drop the entire mathquest-pwa folder contents → Scroll down → Click Commit changes.
💡 Upload everything inside the mathquest-pwa folder — the public/ folder, src/ folder, and package.json file.
Step 2
Deploy on Vercel FREE
Vercel will host your app online with a real URL that anyone can visit.
1
Go to Vercel
Visit vercel.com → Click Sign Up → Choose Continue with GitHub → Authorize Vercel.
2
Import your project
Click Add New → Project → Find your mathquest-ug repository → Click Import.
3
Configure settings
Vercel will auto-detect it's a React app. Leave all settings as default. Just click Deploy.
Vercel will auto-set these for you:
Framework Preset: Create React App Build Command: npm run build Output Directory: build Install Command: npm install
4
Wait ~2 minutes
Vercel builds and deploys your app. When done, you'll get a URL like:
https://mathquest-ug.vercel.app
🎉 Your app is now live on the internet!
5
(Optional) Set a custom domain
In Vercel → Project Settings → Domains → Add your own domain like mathquestug.com. Domains cost ~$10/year from Namecheap or Godaddy.
Step 3
Install on Android Phone
Once your app is live, installing it on Android takes just 3 taps.
🌐
1. Open Chrome
Open Google Chrome on your Android phone and go to your Vercel URL
⋮
2. Tap Menu
Tap the 3-dot menu in the top right corner of Chrome
➕
3. Add to Home Screen
Tap "Add to Home Screen" → Confirm the name → Tap "Add"
🎉
Done!
MathQuest icon appears on your home screen. Open it like any app!
💡 On some Android phones, Chrome shows a banner at the bottom automatically saying "Add MathQuest to Home Screen". Just tap that!
⚠️ Must use Google Chrome (not Samsung Internet or Firefox) for the install prompt to appear reliably.
Step 4
Install on iPhone (iOS)
iPhones use Safari for PWA installation — slightly different steps.
🧭
1. Open Safari
Must use Safari on iPhone — Chrome on iOS cannot install PWAs
📤
2. Tap Share
Tap the Share button at the bottom (the box with an arrow pointing up)
🏠
3. Add to Home Screen
Scroll down in the share sheet and tap "Add to Home Screen"
✅
4. Confirm
Tap "Add" in the top right. MathQuest is now on your iPhone!
Step 5
Share With Students & Schools
Now that your app is live, here's how to get students using it.
1
Share the URL directly
Send your Vercel link via WhatsApp, SMS, or email. Students open it in their browser and install instantly.
Hey! Practice for UCE/UACE for free here 👇 https://mathquest-ug.vercel.app Open in Chrome → tap menu → "Add to Home Screen"
2
Create a WhatsApp group
Create a WhatsApp group for your school or class. Share the link there. Students install and compete on the leaderboard.
3
Pitch to schools
Visit schools and offer a demo. Show the admin dashboard — teachers love being able to see which students need help. Offer school licenses for a monthly fee.
💰 Suggested pricing: Free for first 10 students · UGX 5,000/month per student · UGX 200,000/month school license
